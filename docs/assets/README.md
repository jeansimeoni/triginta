README and user documentation.

- `triginta-icon.png`: square app icon for website or release metadata use
- `github-sponsors.svg`: local README icon for the GitHub Sponsors donation link
- `paypal.svg`: local README icon for the PayPal donation link
- `bitcoin.svg`: local README icon for the Bitcoin donation link
