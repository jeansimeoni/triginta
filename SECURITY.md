# Security Policy

## Supported Versions

Security fixes target the current stable `0.1.x` line and the current `main`
branch.

## Reporting a Vulnerability

Please do not open a public issue for suspected vulnerabilities.

Report security concerns privately to Jean Simeoni through the contact path
available on the project repository or GitHub profile:

- <https://github.com/jeansimeoni/triginta>
- <https://github.com/jeansimeoni>

Include enough detail to reproduce or understand the issue, including affected
versions or commits, operating system, configuration, and any relevant logs with
secrets removed.

## Response Expectations

This is a pre-1.0 project maintained by one person. Security reports will be
triaged as promptly as practical, with priority given to issues that could
expose local data, credentials, or task synchronization state.
